# Events_App

#### Make sure client(port 3000) and server(port 8080) both are running simultaneously to run this project.

#### Server - Port : 8080

1. Navigate to the server directory
   $ cd backend

2. Install dependencies
   $ npm install

3. Run using npm
   $ npm start

#### Client - Port : 3000

1. Navigate to the client directory
   $ cd frontend

2. Install dependencies
   $ npm install

3. Run using npm
   $ npm start
